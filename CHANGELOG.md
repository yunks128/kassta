# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2025

### Added
- React + Vite single-page application with React Router
- Pages: Home, About, Activities, Webinars, Announcements, Leadership, Membership, Donation, Media
- YouTube video integration across pages
- KASSTA Letters section with PDF downloads
- 2025 UKC AeroSpace Forum photo gallery
- Leadership headshot photos
- Partners & Sponsors section with logos and links
- President's Message and bylaws on About page
- Mobile-first responsive CSS design
- Favicon and KASSTA logo branding
- GitHub Pages deployment with custom domain (kassta.ksea.org)
- Open source governance documents (LICENSE, CONTRIBUTING, GOVERNANCE)
- GitHub issue and PR templates
