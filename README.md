# KASSTA Web

Official website for the **Korean American AeroSpace Science and Technology Association (KASSTA)**, a community under [KSEA](https://www.ksea.org/).

Live at: [kassta.ksea.org](https://kassta.ksea.org)

## Tech Stack

- **React 19** with React Router
- **Vite** for development and builds
- Hosted on **GitHub Pages**

## Getting Started

```bash
npm install
npm run dev
```

## Build & Preview

```bash
npm run build
npm run preview
```

## Pages

- Home
- About

- Leadership

## Deployment

The site is automatically deployed to GitHub Pages on push to `main` via GitHub Actions. The custom domain (`kassta.ksea.org`) is configured via the `public/CNAME` file.
